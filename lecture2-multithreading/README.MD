# Приклад з лекції №2 (багатопоточність)
В модулі показано різні способи створення і використання потоків.

В директорії [racecondition](./src/main/java/com/mborodin/javacourse/racecondition)
можна знайти різні реалізації інтерфейсу [Counter](./src/main/java/com/mborodin/javacourse/racecondition/Counter.java)
і порівняти як вони працюють в багатопоточному режимі використовуючи [юніт тести](./src/test/java/com/mborodin/javacourse/racecondition).
