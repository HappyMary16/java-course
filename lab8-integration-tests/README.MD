# Лабораторна робота 8
**Тема**: Інтеграційні тести

**Завдання**: Додати інтеграційні тести до всіх http запитів, які реалізовані в 
`CatController`. Тести потрібно додати в клас `CatControllerTest`, 
в ньому ж можна знайти реалізовані приклади тестів.

Більше прикладів тестів можна подивитись тут:
https://www.baeldung.com/integration-testing-in-spring
